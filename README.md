# 🎓 경기북부 직업교육 학사관리 시스템

엑셀 대신 웹 브라우저로 훈련생·출결·강사·강의실을 한 번에 관리하는 시스템입니다.

---

## ✅ 처음 설치하는 분은 여기서 시작하세요 (3단계)

### 📌 1단계 — Supabase 데이터베이스 만들기

> **Supabase**는 구글처럼 무료로 쓸 수 있는 온라인 데이터 저장소입니다.
> 엑셀 파일 대신 여기에 모든 데이터가 저장됩니다.

1. **[supabase.com](https://supabase.com)** 에 접속해서 회원가입(무료) 후 **새 프로젝트**를 만드세요.
2. 프로젝트가 생성되면 왼쪽 메뉴에서 **"SQL Editor"** 를 클릭하세요.
3. 화면 위쪽 **"+ New query"** 버튼을 클릭하세요.
4. 이 저장소에 있는 **`supabase-setup.sql`** 파일을 열어서 전체 내용을 복사하세요.
5. SQL Editor 입력창에 붙여넣고 **"Run"(또는 Ctrl+Enter)** 을 누르세요.
6. 하단에 **"Success"** 메시지가 나오면 데이터베이스 준비 완료!

> 💡 **이미 예전에 SQL을 실행했는데 저장 오류가 나는 경우:**  
> `supabase-setup.sql`을 다시 한 번 실행하면 됩니다.  
> 파일 끝에 `ALTER TABLE ... ADD COLUMN IF NOT EXISTS` 구문이 있어서  
> **기존 데이터는 유지되면서 누락된 컬럼만 추가**됩니다.

---

### 📌 2단계 — index.html에 내 Supabase 주소 연결하기

> Supabase에서 발급받은 "주소(URL)"와 "비밀번호(Key)"를 `index.html`에 넣어야 합니다.

1. Supabase 프로젝트에서 왼쪽 메뉴 **"Project Settings" → "API"** 로 이동하세요.
2. 아래 두 가지 값을 복사해두세요:
   - **Project URL**: `https://xxxxxxxxxxxx.supabase.co` 같이 생긴 주소
   - **anon public key**: `eyJhbGci...` 로 시작하는 긴 문자열

3. 이 저장소의 **`index.html`** 파일을 메모장(또는 VSCode)으로 열고,
   **33번째 줄** 근처를 찾으세요:

   ```
   // ↓↓↓ 여기에 내 Supabase 정보를 입력하세요 ↓↓↓
   const SB_URL = "https://여기에-Project-URL-입력";
   const SB_KEY = "여기에-anon-public-key-입력";
   // ↑↑↑ 입력 완료 후 저장하세요 ↑↑↑
   ```

4. `"https://여기에-Project-URL-입력"` 부분을 Supabase Project URL로 교체하세요.
5. `"여기에-anon-public-key-입력"` 부분을 anon public key로 교체하세요.
6. 파일을 **저장(Ctrl+S)** 하세요.

---

### 📌 3단계 — 브라우저에서 실행하기

1. `index.html` 파일을 **크롬(Chrome) 브라우저**로 여세요.  
   (파일을 더블클릭하거나, 크롬에 드래그&드롭 하면 됩니다)
2. 로그인 화면이 나타납니다.
3. 아이디: **관리자**, 비밀번호: **admin1234** 로 로그인하세요.
4. 🎉 완료! 모든 탭을 자유롭게 사용하세요.

## ❓ 자주 묻는 질문

**Q. "DB 연결 오류"가 뜹니다.**  
A. `index.html` 33~34번째 줄의 SB_URL, SB_KEY가 올바른지 확인하세요.  
   그리고 `supabase-setup.sql`을 Supabase에서 실행했는지 확인하세요.

**Q. "Could not find the '...' column" 저장 오류가 납니다.**  
A. 이전 버전 SQL로 만든 DB에 새 컬럼이 없거나, Supabase 스키마 캐시가 갱신되지 않은 경우입니다.  
   다음 순서대로 시도해보세요:

1. `supabase-setup.sql`을 **다시 한 번 실행**하세요 — 기존 데이터는 유지되면서 누락된 컬럼이 추가됩니다.  
2. 그래도 같은 오류가 나면: Supabase 대시보드 → **Settings(설정) → API** 메뉴로 이동하여 **"Reload schema"** 버튼을 클릭하세요.  
   (Supabase가 이전 테이블 구조를 기억하고 있을 때 발생합니다)

**Q. 로그인이 안 됩니다.**  
A. `supabase-setup.sql` 실행 후 accounts 테이블에 관리자 계정이 생성됩니다.  
   아이디 `관리자`, 비밀번호 `admin1234`로 로그인하세요.

**Q. 출결 QR코드가 작동 안 합니다.**  
A. 모바일에서 QR을 스캔하면 됩니다. 데스크탑 브라우저에서 직접 스캔은 불가합니다.

**Q. Supabase는 유료인가요?**  
A. 월 5만 행 이하는 완전 무료입니다. 일반 행정 업무에는 충분합니다.
